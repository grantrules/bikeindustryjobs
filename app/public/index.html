careers.bike

Hi, I'm Grant, I created this site as a personal project during my job search for a tech job in the bike industry. Jobs would be listed on company sites, LinkedIn, Indeed, and other sites, this site aims to simplify the search for a career in the bike industry by aggregating those job listings!

I wrote this in Javascript using Mongo, Node, Express, and React. Search is powered by Bloodhound. Check out the project on GitHub

I spent the last 7 years promoting cycling at one of the best-reviewed bike shops in Brooklyn and before that I worked as a Java and PHP developer at a financial tech startup. I'm really hoping to combine those two experiences into a tech career in the cycling industry.
